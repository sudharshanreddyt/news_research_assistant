End-to-end LLM project for beginners and intermediate users using langchain. 
In this LLM project, we will use langchain, openai API, and streamlit to build a news research tool that can be used by equity research analysts to conduct their research. 
This end-to-end NLP project will give you a good experience in building a real-life industry project that will add a lot of value to your data scientist or NLP engineer project portfolio.
